# Neovim 使い方ガイド

## 設定完了！ まず試してみよう

```bash
nvim /tmp/nvim-test.txt
```

このファイルを開いたら：
1. **そのまま（ノーマルモード）** で `Space` + `e` を押す → ファイルツリーが表示
2. `i` を押す → インサートモードで文字入力
3. `Esc` を押す → ノーマルモードに戻る
4. `:q` + Enter → Neovimを終了

# Neovim 使い方ガイド

## モードの概念

Neovimには3つの主要なモードがあります：

### 1. ノーマルモード（Normal Mode）
- 起動直後のモード
- **ここでキーマップを使う**
- 左下に何も表示されていない = ノーマルモード
- 文字入力はできない

### 2. インサートモード（Insert Mode）
- 文字を入力するモード
- 左下に `-- INSERT --` と表示される
- `i` キーで切り替え
- `Esc` でノーマルモードに戻る

### 3. ビジュアルモード（Visual Mode）
- テキスト選択モード
- `v` キーで開始

---

## 基本的な流れ

```
1. nvim を起動
   ↓
2. [ノーマルモード] ← ここでコマンド実行
   ↓
3. i を押す
   ↓
4. [インサートモード] ← ここで文字入力
   ↓
5. Esc を押す
   ↓
6. [ノーマルモード] に戻る
```

---

## よく使うキーマップ（ノーマルモードで使用）

### ファイル操作
- `Space + e` → ファイルツリー表示/非表示
- `Space + p` → ファイル検索（Telescope）
- `Space + g` → 文字列検索（Grep）

### ウィンドウ操作
- `Space + s` → 水平分割
- `Space + v` → 垂直分割
- `Space + h/j/k/l` → ウィンドウ間移動

### 保存・終了
- `Space + w` → 保存
- `Space + q` → 閉じる
- `Space + wq` → 保存して閉じる

### 編集
- `i` → カーソル位置から入力開始
  `a` → カーソルの次から入力開始
- `o` → 下に新しい行を作成して入力
- `Esc` → ノーマルモードに戻る

### LSP（コード補完・エラー検出）
- `gd` → 定義へジャンプ
- `gr` → 参照を表示
- `Space + ca` → コードアクション
- `Space + d` → エラー表示

### フォーマット
- `Space + i` → コードフォーマット実行

---

## 実際に試してみよう

1. この画面（ノーマルモード）で `Space + e` を押す
   → 左側にファイルツリーが表示される

2. `Space + p` を押す
   → ファイル検索が開く（文字を入力して検索）

3. `i` を押す
   → インサートモードになり、文字が入力できる

4. 何か文字を入力する

5. `Esc` を押す
   → ノーマルモードに戻る

6. `Space + w` を押す
   → ファイルを保存

---

## トラブルシューティング

### キーマップが動かない
- ノーマルモードにいるか確認（左下に何も表示されていない）
- インサートモードなら `Esc` を押す

### プラグインが動かない
- `:Lazy` と入力してEnter → プラグイン管理画面を開く
- `S` キーでプラグインを同期（Sync）

### 終了できない
- `Esc` を何度か押してノーマルモードに戻る
- `:q!` と入力してEnter（保存せず強制終了）

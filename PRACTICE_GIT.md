# Git操作の練習ファイル

このファイルで実際にgitsignsの操作を試してみましょう。

## ステップ1: 変更箇所の確認

以下の行を編集してみてください：
- この行を変更してみる
- この行も変更してみる
- 新しい行を追加してみる

## ステップ2: 変更箇所の移動

編集が終わったら、以下のキーで変更箇所を移動できます：
- ]c : 次の変更箇所へ
- [c : 前の変更箇所へ

## ステップ3: 変更のプレビュー

変更箇所にカーソルを置いて：
- Space + hp : 変更内容をポップアップで確認

## ステップ4: ステージング

気に入った変更をステージング：
- Space + hs : カーソル行をステージング
- Space + hS : ファイル全体をステージング

## ステップ5: 変更を元に戻す

やっぱり要らない変更だった場合：
- Space + hr : カーソル行の変更を元に戻す
- Space + hR : ファイル全体を元に戻す

## よく使う操作まとめ

| キー | 動作 |
|------|------|
| ]c | 次の変更箇所へ |
| [c | 前の変更箇所へ |
| Space + hp | 変更をプレビュー |
| Space + hs | 行をステージング |
| Space + hr | 変更を元に戻す |
| Space + hb | Blame表示 |

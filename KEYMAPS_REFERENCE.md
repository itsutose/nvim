# 現在のキーマップ一覧

## 🎯 基本操作

### モード切り替え
| キー | 動作 | モード |
|------|------|--------|
| `i` | インサートモードに入る | ノーマル |
| `Esc` または `jj` | ノーマルモードに戻る | インサート |
| `v` | ビジュアルモード | ノーマル |

---

## 📁 ファイル・ウィンドウ操作

### ウィンドウ移動（keymaps.lua）
| キー | 動作 |
|------|------|
| `Shift + H` | 行頭へ移動 |
| `Shift + L` | 行末へ移動 |
| `Ctrl + h` | 行頭へ移動 |
| `Ctrl + ;` | 行末へ移動 |

### コメント
| キー | 動作 |
|------|------|
| `Cmd + /` | コメントトグル |

---

## 🔀 Git操作（gitsigns.nvim）

### 変更箇所の移動
| キー | 動作 |
|------|------|
| `]c` | 次の変更箇所へ |
| `[c` | 前の変更箇所へ |

### Git操作（Space + h + キー）
| キー | 動作 |
|------|------|
| `Space + hs` | ステージング（現在行） |
| `Space + hu` | アンステージング |
| `Space + hr` | 変更を元に戻す（現在行） |
| `Space + hp` | 変更をプレビュー |
| `Space + hb` | Blame表示 |
| `Space + hd` | 差分表示 |
| `Space + hS` | ファイル全体をステージング |
| `Space + hR` | ファイル全体を元に戻す |
| `Space + tb` | Blame自動表示ON/OFF |

### LazyGit
| キー | 動作 |
|------|------|
| `Space + gg` | LazyGit起動 |
| `Space + gf` | 現在ファイルのGit履歴 |

---

## 🔍 Neovimでキーマップを確認する方法

### コマンドライン
```vim
:map           " すべてのマップ
:map <leader>  " Spaceキーから始まるマップ
:nmap          " ノーマルモードのマップ
:vmap          " ビジュアルモードのマップ
:imap          " インサートモードのマップ

" 特定のキーを検索
:map <C-h>     " Ctrl+hのマップを確認
```

### プラグインのヘルプ
```vim
:help gitsigns
:help lazygit
```

## 💡 おすすめ：which-key.nvimを追加

which-key.nvimを追加すると、キーを押すだけで候補が表示されます。

**メリット:**
- キーを押して少し待つだけで候補が表示
- 説明付きで表示
- キーマップを覚える必要なし

**デメリット:**
- 設定ファイルが1つ増える
- 少しだけ遅延が発生（カスタマイズ可能）
